STEP 01/09

Canyon 接続ガイド

このガイドでは、Gitサーバー「Canyon」に接続するために必要な設定を順番に案内します。
所要時間は初回のみ15〜30分ほどです。

Canyon とは？

「Canyon」は、モデリングコミュニティ向けに用意された非公開のGitサーバーです。
大容量ファイルの管理（Git LFS対応予定）や、メンバー間のプロジェクト共有が行えます。
サーバーはTailscale VPN経由でのみアクセス可能なため、外部からの不正アクセスは遮断されています。

Step 1: 招待を受け取る

Canyonへのアクセスには、管理者からTailscaleの招待が必要です。
管理者に連絡して、招待メールを送ってもらってください。

招待メールの送信元: Tailscale <no-reply@tailscale.com>
メール内の「Accept invite」または「招待を承諾する」をクリックしてください。
招待が届かない場合
  • 迷惑メールフォルダを確認してください。
  • 管理者に再送を依頼してください。
  • すでにTailscaleアカウントをお持ちの場合、招待はそのアカウントに関連付けられます。

Step 2: Tailscale のインストールと接続

Tailscaleをインストールし、招待を承諾したアカウントでサインインしてください。
アプリのアイコンが「Connected」になれば、サーバーへのアクセスが可能になります。

OS別ダウンロード先
  • Windows: tailscale.com/download/windows ↗
  • macOS: App Store で「Tailscale」を検索、または tailscale.com/download/mac ↗
  • Linux: curl -fsSL https://tailscale.com/install.sh | sh

救済対応: Canyon Bridge (Docker)

もし公式のTailscaleアプリでの接続がうまくいかない場合は、この救済キットを使ってください。
Docker版キットを使うことで、ネットワーク環境に左右されず強制的に接続できます。

1. 管理者から配布された canyon-bridge.zip を解凍。
2. .env.example を .env にして認証キーを入力。
3. docker compose up -d を実行。

セットアップ後は http://localhost:3000 でGiteaにアクセスできます！

Step 3: Gitea アカウントの作成

Gitサーバーの操作画面（Gitea）にアクセスして、ユーザーアカウントを作成してください。
Gitea を開く ↗

アカウント作成の手順
  1. 上のリンクからGiteaのトップページを開く。
  2. 右上の「Register（登録）」をクリック。
  3. ユーザー名・メールアドレス・パスワードを入力して登録する。
  4. 登録後、管理者にユーザー名を伝えると、リポジトリへのアクセス権が付与されます。

Step 4: SSH 認証鍵の生成

Gitの操作（clone / push / pull）にはSSH鍵認証が必要です。
下のボタンを押すと、ブラウザ内で鍵ペア（公開鍵・秘密鍵）が生成されます。鍵はサーバーに送信されません。

SSH鍵とは？秘密鍵の保存場所について

SSH鍵はパスワードの代わりに使用する認証情報です。「秘密鍵」と「公開鍵」の2つがセットです。

  • 公開鍵（次のステップでGiteaに登録）: サーバーに渡すもの。外部に公開しても問題ありません。
  • 秘密鍵（ダウンロードして保存）: 絶対に他人に渡さないでください。

秘密鍵の保存先:

  • Windows: C:\Users\あなたのユーザー名\.ssh\id_ed25519
  • macOS / Linux: ~/.ssh/id_ed25519

保存後、chmod 600 ~/.ssh/id_ed25519（macOS/Linux）で権限を設定してください。

Step 5: 公開鍵を Gitea に登録する

前のステップでコピーした公開鍵を、Giteaのアカウント設定に登録してください。
Gitea → SSH鍵の設定ページを開く ↗

登録手順
  1. 上のリンクからGiteaのSSH鍵設定ページを開く。
  2. 「鍵を追加する」または「Add Key」をクリック。
  3. 「タイトル」に任意の名前（例: my-pc）を入力。
  4. 「コンテンツ」欄に、前のステップでコピーした公開鍵（ssh-ed25519 AAAA... で始まる文字列）を貼り付ける。
  5. 「鍵を追加する」ボタンで保存する。

Step 6: TortoiseGit のインストール

Windowsで Git を GUI で操作するためのツール「TortoiseGit」をインストールしてください。
右クリックメニューからclone・push・pullができるようになります。

🐢 TortoiseGit をダウンロード ↗
インストール手順
  1. 上のリンクから TortoiseGit（64-bit）をダウンロードしてインストールする。
  2. インストール後、任意のフォルダを右クリックして「TortoiseGit」メニューが表示されることを確認する。
  3. 初回起動時に「Git for Windows」が必要と案内される場合は、同ページから Git for Windows もインストールする。
SSH設定ファイルの確認（重要）

Step 4 でダウンロードした config ファイルを、以下の場所に配置してください。
このファイルにより、TortoiseGit が Canyon の特殊なSSHポート（222番）を自動認識します。

  • 配置先: C:\Users\あなたのユーザー名\.ssh\config
  • （.ssh フォルダがない場合は新規作成してください）

ファイルの中身はこのようになっています：

Host canyon
    HostName canyon-gitea.tail2582f9.ts.net
    Port 222
    User git
    IdentityFile ~/.ssh/id_ed25519

Step 7: TortoiseGit でクローンする

TortoiseGit を使って、リポジトリをPCにダウンロード（クローン）します。
(Org) と (Repo) は管理者に確認してください。

canyon:/(Org)/(Repo).git
クローン手順（TortoiseGit）
  1. クローン先にしたいフォルダを右クリックし、「TortoiseGit」→「クローン...」を選択。
  2. 「URL」欄に上のURL（canyon:/... の形式）を入力する。
  3. 「参照するディレクトリ」に保存したいフォルダのパスを確認して「OK」。
  4. 初回接続時に「Known hosts」への追加確認ダイアログが出たら「はい」を選択。
  5. クローンが完了したら成功です。
クローンが失敗する場合
  • Tailscaleが「Connected」状態か確認してください。
  • C:\Users\あなた\.ssh\config が正しく配置されているか確認してください。
  • C:\Users\あなた\.ssh\id_ed25519（秘密鍵）が存在するか確認してください。
  • GiteaのSSH設定に公開鍵が登録されているか確認してください（Step 5）。
  • Giteaのリポジトリへのアクセス権が付与されているか管理者に確認してください。
